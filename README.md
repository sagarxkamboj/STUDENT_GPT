# 🎓 StudentGPT – Student Resource Portal

🌐 **Live Website:** https://studentgptsagar.netlify.app/

---

## 📌 About the Project

StudentGPT is a student-focused web platform designed to provide useful academic resources, tools, and information for students in a simple and organized way.

The goal of this project is to make study materials easily accessible while also demonstrating modern web development skills such as responsive design, clean UI, and web deployment.

This project serves as both a **learning platform for students** and a **portfolio project showcasing development skills**.

---

## ✨ Features

* 📚 Easy access to student resources
* 💻 Clean and simple user interface
* 📱 Fully responsive design (works on mobile & desktop)
* ⚡ Fast loading web pages
* 🌐 Hosted online using Netlify
* 🧑‍💻 Beginner-friendly web project

---

## 🛠 Technologies Used

* HTML5
* CSS3
* JavaScript
* Git & GitHub
* Netlify (Hosting & Deployment)

---

## 🚀 How to Run the Project Locally

1. Clone the repository

2. Open the project folder

3. Run the project by opening **index.html** in your browser
